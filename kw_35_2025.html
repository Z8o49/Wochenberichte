HTML und CSS Basics

Ein Bild mit dem HTML und CSS Logo auf weissem Hintergrund. Quelle: Google

HTML (HyperText Markup Language) bildet die Grundstruktur einer Webseite, also Überschriften, Texte, Bilder oder Links. CSS (Cascading Style Sheets) sorgt dafür, dass diese Elemente auch gut aussehen, zum Beispiel durch Farben, Schriftarten oder Abstände. Man kann sich HTML wie das Gerüst eines Hauses vorstellen, CSS wie die Einrichtung und Dekoration.

HTML

HTML (HyperText Markup Language) wurde Ende der 1980er Jahre von Tim Berners-Lee am CERN entwickelt. Ziel war es, Texte und Informationen im Internet strukturiert darzustellen und über Links miteinander zu verknüpfen, sodass Benutzer einfach zwischen Dokumenten im World Wide Web navigieren können.

HTML arbeitet mit sogenannten „Elementen“ oder „Tags“, die Inhalte oder Strukturen definieren. Beispiele sind <h1> bis <h6> für Überschriften, <p> für Absätze, <ul>/<ol> für Listen oder <img> für Bilder. Jedes Element kann Attribute besitzen, die zusätzliche Informationen liefern, wie Grösse, Links oder Formatierungen. HTML bildet das Grundgerüst einer Webseite und legt fest, welche Inhalte angezeigt werden und in welcher Reihenfolge sie erscheinen.

CSS

CSS (Cascading Style Sheets) wurde Mitte der 1990er Jahre eingeführt, um Webseiten optisch zu gestalten und von der HTML-Struktur zu trennen. Während HTML nur die Struktur liefert, sollte CSS das Aussehen konsistent und flexibel definieren.

CSS arbeitet über Regeln, die HTML-Elementen zugewiesen werden. Damit lassen sich Farben, Schriftarten, Abstände, Grössen, Rahmen und Layouts festlegen. Diese Regeln können direkt im HTML-Code, in <style>-Blöcken oder in externen CSS-Dateien stehen. CSS ermöglicht es, das Design einer gesamten Webseite einheitlich zu gestalten, ohne die HTML-Struktur zu verändern. So „kleidet“ CSS das HTML-Grundgerüst ein und sorgt für ein ansprechendes Layout.

Fazit

HTML und CSS bilden zusammen die Grundlage jeder Webseite. HTML sorgt für die Struktur, während CSS das Aussehen gestaltet. Auch wenn wir erst am Anfang stehen, habe ich schon einigermassen verstanden, wie beide zusammenarbeiten und miteinander ein Programm ergeben.
